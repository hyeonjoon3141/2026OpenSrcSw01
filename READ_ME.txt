git init : 깃 초기화
git status : 깃 상태 확인
git add [FILE] : 지정 파일 Staging Area에 추가
git commit -m "MESSAGE" : 커밋 저장 (-m : 메시지 옵션) 
git log : 커밋 기록 확인 (최신순 정렬)

git 프로필 설정

git config --global user.name "NAME"
git config --global user.email "EMAIL"
git config --list : 리스트 확인

git폴더 안에서 ls -al 하면 .git 폴더가 있음 (그냥 ls하면 안보임)
